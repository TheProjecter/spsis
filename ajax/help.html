How To's

  • Spare Part/Supply
    • Add Spare Part/Supply
      -> Find tab for Spare Part/Supply
      -> Click Add Item
    • View Spare Part/Supply Information
      -> Find Home tab
      -> Click View of spare part/supply of interest
    • Edit Spare Part/Supply
      -> Find Home tab
      -> Click View of spare part/supply of interest
      -> Click Edit



  • Machine
    • Add Machine
      -> Find Machine tab
      -> Click Add Machine
    • View Machine Spare Parts
      -> Find Machine tab
      -> Click List of Machines
      -> Click Machine of interest
    • Delete Machine
      -> Find Machine tab
      -> Click List of Machines
      -> Click Machine of interest
      -> Click Delete
  • Account
    • View Account
      -> Find Account tab
      -> Click View My Account
    • Edit Account
      -> Find Account tab
      -> Click Edit My Account
    • Search Account
      -> Find Account tab
      -> Click Search Account

      -> Find Search tab
      -> Click User Account
